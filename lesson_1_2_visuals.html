Lesson 1.2: Stakeholder Mapping & Engagement

Animated Visual Guide

Core Framework Visuals

1. Stakeholder Mapping Framework - Three-Layer System

Core Framework: The three-layer stakeholder mapping system showing Primary (directly affected), Secondary (influence & expertise), and Tertiary (indirect interest) stakeholders with visual hierarchy through size and positioning.

2. Power-Interest Analysis Grid

Power-Interest Grid: The foundational analysis tool showing four engagement strategies based on stakeholder power and interest levels, with animated stakeholder placement and strategy labels.

Process Flow Visuals

3. Problem Tree to Stakeholder Engagement Flow

Process Flow: Shows how Problem Tree assumptions (A) are validated through stakeholder engagement to create evidence-based (E) refined analysis, demonstrating the transformation from preliminary to community-grounded understanding.

4. Engagement Strategy by Stakeholder Type

Engagement Strategy: Demonstrates differentiated engagement approaches for each stakeholder category, showing how methods, timing, and depth of engagement adapt to stakeholder characteristics and capacity.

Supporting Process Visuals

5. Successful vs Unsuccessful Engagement

Engagement Comparison: Visual contrast between projects that skip stakeholder engagement (leading to failure) versus those that prioritize community collaboration (leading to success).

6. Best Practices Checklist

Best Practices: Animated checklist showing the six core principles for meaningful stakeholder engagement, with progressive reveal to emphasize each practice.

7. Integration Back to Problem Tree

Integration Process: Shows how stakeholder insights flow back into the Problem Tree, converting assumptions to evidence (E tags) and adding new elements discovered through community engagement.

8. Community Settings for Engagement

Community Settings: Illustrates the diversity of appropriate engagement settings, emphasizing the importance of meeting stakeholders in familiar, comfortable, and culturally appropriate spaces.

Visual Learning Summary

These animated visuals support the core learning objectives of Lesson 1.2 by illustrating the systematic approach to stakeholder mapping and engagement. Each visual reinforces key concepts through progressive animation that mirrors the step-by-step methodology taught in the lesson.
